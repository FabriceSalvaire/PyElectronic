2.1.6. ResistorColourCode¶

This modules provides tools to work with resistor colour code and the EIA (Electronics Industries Association) resistor series values.

The standardised E6, E12, E24, E48, E96 and E192 resistor series values are made available through an instance of the class ValuesSeries named E6 and so on.

PyResistorColorCode.ResistorColourCode.COLOUR_CODES = {'silver': silver: x10 m, 10.00 %, 'gold': gold : x100 m, 5.00 %, 'black': black : d0, x1, 250 ppm, 'brown': brown : d1, x10, 1.00 %, 100 ppm, 'red': red : d2, x100, 2.00 %, 50 ppm, 'orange': orange: d3, x1 k, 15 ppm, 'yellow': yellow: d4, x10 k, 5.00 %, 25 ppm, 'green': green : d5, x100 k, 0.50 %, 20 ppm, 'blue': blue : d6, x1 M, 0.25 %, 10 ppm, 'violet': violet: d7, x10 M, 0.10 %, 5 ppm, 'grey': grey : d8, x100 M, 0.05 %, 1 ppm, 'white': white : d9, x1 G}¶

This dictionnary maps the colour name and their ColourCode instance.

PyResistorColorCode.ResistorColourCode.COLOUR_NAMES = ('silver', 'gold', 'black', 'brown', 'red', 'orange', 'yellow', 'green', 'blue', 'violet', 'grey', 'white')¶

This list defines the ordered set of colours.

class PyResistorColorCode.ResistorColourCode.ResistorDecoder[source]¶

Bases: object

This class implements a resistor decoder using an inference algorithm:

  • The given list of colours doesn’t require to be oriented (code polarity), both orientation Right-Left and Left-Right are tested (bidirectional inference),
  • At least three colours must be provided (two digits colour and the multiplier colour),
  • The colours are interpreted by priority as resistance value, then tolerance and finally temperature coefficient.
  • The resistance value must exists in a series and its tolerance must be defined if there is a colour assigned to it.

class PyResistorColorCode.ResistorColourCode.ValuesSeries(name, number_of_digits, tolerances, values)[source]¶

Bases: object

This class defines the properties of resistor value series like the standardised E6, E12, E24, E48, E96 and E192 resistor series values.

The parameter name gives the name of the series, the parameter number_of_digits defines the number of digits of the values, the parameter tolerances gives the list of tolerances and values gives the series of values.

Public attributes:

name

number_of_digits

tolerances

values

The name of the series can be retrieved using:

str(E6)

The number of values is given by:

len(E6)

We can test if a value is in the series using:

10 in E6

We can compare two series using the order relation on their number of values, for example E6 < E12 is True.
