2.1.8. ResistorWidget¶

This modules provides GUI widgets for resistors.

class PyResistorColorCode.ResistorWidget.ColourMatrix(parent)[source]¶

Bases: PyQt5.QtCore.QObject

This class implements a colour matrix where the columns represents the resistor colour bands and the rows all the possible colours and the undefined state. Each band column is made of a group of exclusive checkable buttons.
