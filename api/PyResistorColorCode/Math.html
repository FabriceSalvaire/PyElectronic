3.1.5. Math¶

This module provides math tools.

PyResistorColorCode.Math.number_of_digits_of(x)[source]¶

Return the number of digits of the integer part of x.

Let x an integer \(\geq 1\), we could rewrite it as a normalised form \(0.x\ 10^n\), where n are the number of digits of x. The logarithm in base 10 of x is equal to \(n + \log_{10}(0.x)\). Since \(0.1 \leq 0.x < 1\), we have \(-1 \leq \log_{10}(0.x) < 0\) and thus \(n \leq \log_{10}(x) + 1 < n + 1\). It follows the number of digits of x is equal to \(int(\log_{10}(x) + 1)\).

PyResistorColorCode.Math.significant_digits_of(x, number_of_significant_digits)[source]¶

Return the most significant digits of a number, for example:

significant_digits_of(1234, 2)

will return 12.
